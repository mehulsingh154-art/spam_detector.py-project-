# Spam Email Detection using Machine Learning

## Problem Statement
Spam messages are a common issue in communication systems. This project aims to detect whether a message is spam or not using Machine Learning.

## Objective
To classify messages into Spam or Not Spam using a trained model.

## Approach
- Used Naive Bayes Algorithm
- Converted text into numerical form using CountVectorizer
- Trained model on sample dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn

##  How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python spam_detector.py

##  Output
The system takes a message as input and predicts whether it is spam or not.
